que es una aplicacion?
podria ser, una herramienta desarrollada para darle alguna utilidad. (redes sociales.. juegos.. etc). El objetivo! hacer aplicaciones para la web (ya fue dicho antes pero para recalcar.)

desktop -> c#, electron (apps para desktop)

un navegador (browser) tiene una barra de direcciones que se traslada a una pagina.
una app es web por el echo de que se abre desde un navegador.

al momento de introducir una direccion en el navegador, la aplicacion esta guardada en una nube, la cual reedirige la direccion (o peticion) que se realizo a un servidor, luego este ultimo devuelve el contenido a la nube ,y de la nube al pc o navegador

http= protocolo. es como se van a comunicar el navegador y el servidor.
nube= internet.

una aplicacion esta conformado por un cliente, una peticion, un protocolo y servidor.
